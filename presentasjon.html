Tjeneste 1 & 2 av 12

KAS Core & Sales Core

Backend · Node.js · TypeScript
🏗️

KAS Core

Kilde til sannhet

🎯 Ansvar

  • Master-API for alle kunder og prospekter
  • Produktkatalog — SDU og MDU pakker
  • Kampanjebibliotek med prislogikk
  • Interessescore-beregning per beboer

Nøkkelpunkt

Endrer du en kampanjepris i KAS Core er det oppdatert i alle apper umiddelbart — ingen frontend-deploy nødvendig.

📋

Sales Core

Selgerregister

🎯 Ansvar

  • Selgerregister med roller og tilganger
  • Rundeplanlegging og tildeling
  • Besøksstatus per enhet (pending → sold)
  • Statistikk: konvertering og effektivitet

Separasjon av ansvar

Sales Core håndterer hvem som selger hva og når. KAS Core håndterer hvem kunden er og hvilke produkter som finnes.

Besøksflyt

pending → visited → sold

Tjeneste 5 & 6 av 12

MDU Leder & Salgs Hub

📊

MDU Leder

Ledelses­dashboard
  • Nøkkeltall: pipeline-verdi, konvertering, aktivitet
  • Teamoversikt med status per selger
  • War Room — godkjenn og avvis store deals
  • AI Sidekick med live pipeline-kontekst

Leder ser hva hvert teammedlem jobber med i sanntid, og kan justere kampanjer uten å involvere IT. AI Sidekick svarer på pipeline­spørsmål direkte fra dashboardet.

🚀

Salgs Hub

Gateway
  • Én innloggingsflate for alle roller
  • Roling­basert visning: riktig app til riktig person
  • SSO-sesjon videresendt som signert token
  • Innsikts­agent og NBA-agent for ledelsen

Gateway-mønsteret

Salgs Hub er den eneste appen brukeren trenger å huske. Den distribuerer sesjonen til alle andre apper — og hoster AI-agentene.

AI-laget · Tjeneste 10, 11 & 12 av 12

Kunstig intelligens i hvert steg av salget

🧠

AI Core

Backend · Claude claude-opus-4-6

  • NBA — Next Best Action for feltselger
  • Kontekst­bevisst Sidekick API
  • Tverr­kanal innsikts­analyse
  • Lærer av hvert salg og utfall

Alle AI-kall går gjennom AI Core — én tjeneste, én API-nøkkel, full kontroll og logging.

💬

AI Sidekick

Flytende chatbot · Alle apper

  • Selger-modus: neste steg, oppfølging, innsigelse­håndtering
  • Leder-modus: pipeline­overblikk, risk­analyse, statusrapport
  • Kontekst fra live pipeline injiseres automatisk
  • Skriver e-poster og oppsummeringer

Leder spør «Hvilke deals er i fare?» — Sidekick svarer med live data fra War Room.

🔍

Hub Agenter

Innsikts­agent + NBA-agent

  • Analyserer salgsdata på tvers av SDU og MDU
  • Churn­årsaker, win­faktorer, produkt­trender
  • NBA-ytelse: treffsikkerhet, divergens­par, topp­produkter
  • Delt læring mellom kanaler

Én klikk — AI analyserer alle salg og returnerer strukturert rapport med anbefalinger.

Arkitekturvalg

Gateway-mønster, modulær arkitektur & AI-fremtid

🚪 Hvorfor gateway?

  • → Én SSO-inngang — brukere logger inn ett sted
  • → Rollestyring sentralisert, ikke spredt i appene
  • → Nye apper kan legges til uten å endre eksisterende

🧩 Hvorfor modulær?

  • → SDU og MDU kan itereres uavhengig
  • → En feil i ett system stopper ikke resten
  • → Skalerbar etter behov

Production Readiness

Fra PoC til enterprise-grade

🔐 Sikkerhet & Identitet

  • Koble til eksisterende SSO (Azure AD / Entra ID)
  • JWT-validering i alle backend-API-er
  • Scope-basert tilgangskontroll (RBAC)
  • TLS overalt — allerede håndtert av hosting
  • API-nøkler og hemmeligheter i Azure Key Vault
  • Revisjonslogg på alle datamutasjoner

🗄️ Data & Hosting

  • Flytt til Azure Norway East — allerede sikker infrastruktur
  • PostgreSQL erstatter in-memory data
  • Daglig backup og point-in-time recovery
  • Data-residency: Norge/EU for GDPR
  • Private nettverk mellom tjenestene (VNet)
  • CDN for frontend-appene

📈 Drift & Kvalitet

  • CI/CD pipeline med automatiske tester
  • Overvåking og alerting (Datadog / Azure Monitor)
  • Structured logging og distribuert sporing
  • Rate limiting og DDoS-beskyttelse
  • Staging-miljø — separat fra prod
  • SLA-definisjon og on-call rutiner

Nøkkelpunkt: Arkitekturen er allerede cloud-native og stateless — det meste av produksjons­tilpasning handler om å koble til eksisterende infrastruktur (SSO, database, hosting) som Telenor allerede har på plass.

Anbefalt tilnærming
